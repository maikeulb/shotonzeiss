# ShotOnZeiss

Simple social network (similar to Flickr) for Zeiss lovers. Users share their
favorite photos taken with Zeiss lenses.

Technology
----------
* React
* Redux + Redux-Thunk
* Ant-Design
* Styled-Components

Infrastructure
----------
* Firebase Realtime Database
* Firebase Storage
* Firebase Authentication
* Firebase Hosting
